# Tugas-3-Proyek-Pemrograman-Web
program ini akan meminta user menginput tanggal dan bulan lahir untuk menentukan zodiak mereka.
untuk membuat user bisa menginput, program ini menggunakan readline.
lalu digunakan kode let untuk menyimpan hasil perhitungan.
menggunakan if else untuk mengecek dan menentukan zodiak yang sesuai dengan input yang diisi user
lalu menampilkan ouput menggunakan console.log, disini akan dipanggil variabel tanggal, bulan dan juga zodiak yang telah ditentukan.
